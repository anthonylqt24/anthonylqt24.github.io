Some highlight school projects

Bank Term Deposit Subscription Projection

Link to AI project's GitHub repo

This is the final project that I work with my classmates in a team of five. A Python program that utilized pandas, scikit-learn, tensorflow, and numpy to articulate our data to achieve the result of projecting a probability of whether a customer will subscribe to a term deposit at a bank or not. Three models was used including Multi-layer Perceptron, Gaussian Naive Bayes, and K-Means.

Caro Game with AI player using Rust

Link to caro project's GitHub repo

The game was written in Rust which is a fun language. It will correct the bugs you might have in your code and give suggestions on how your code should be written. We decided to recreate the game called "caro" which is an expanded version of tic-tac-toe. You have to match five instead of three. We made an AI player to compete against human. We used Iced version 0.3.0 cross-platform GUI library with a MIT license.
