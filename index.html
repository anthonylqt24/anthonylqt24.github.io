Some highlight school projects

The best way to contact me

Here's the GitHub repo of my project. I am going to use a project from my Machine Learning course. This is the final project that I work with my classmates in a team of five. The language that we chose to work on was Python. We also used libraries such as pandas, scikit-learn, tensorflow, and numpy to articulate our data to achieve the result of projecting a probability of whether a customer will subscribe to a term deposit at a bank or not. We used data from a Portuguese banking institution to do the prediction. The main reason we built this project was to assess the machine learning algorithms that we learned in the course with the real life data. We wanted to measure the accuracy of them and to empower ourselves with new technologies that help increase the process of big data calculation. We used three models including Multi-layer Perceptron, Gaussian Naive Bayes, and K-Means. The results were very fascinating. We did our best to minimize the errors and decrease the runtime of the project.
