Some highlight school projects

Bank Term Deposit Subscription Projection

Link to AI project's GitHub repo

This is the final project that I work with my classmates in a team of five. The language that we chose to work on was Python. We also used libraries such as pandas, scikit-learn, tensorflow, and numpy to articulate our data to achieve the result of projecting a probability of whether a customer will subscribe to a term deposit at a bank or not. We used data from a Portuguese banking institution to do the prediction. The main reason we built this project was to assess the machine learning algorithms that we learned in the course with the real life data. We wanted to measure the accuracy of them and to empower ourselves with new technologies that help increase the process of big data calculation. We used three models including Multi-layer Perceptron, Gaussian Naive Bayes, and K-Means. The results were very fascinating. We did our best to minimize the errors and decrease the runtime of the project.

Caro Game with AI player using Rust

Link to caro project's GitHub repo

The game was written in a new language that none of us had any knowledge before. Rust is a fun language to learn and program. It will correct the bugs you might have in your code and give suggestions on how your code should be written. The project was a substitute for our final in this course. We decided to recreate the game in our childhood. It is called "caro" which basically is an expanded version of tic-tac-toe. But instead of matching three, you have to match five of the same symbol. We made an AI player to compete against human player. One of our friend tried the game.We made it so smart that it won 5/5 matches with him. The game was a quite success for the AI also for the new language that we only have a chance to learn in 2 months. We used a Rust GUI library, Iced version 0.3.0 cross-platform GUI library inspired by Elm. The game is in MIT license and was built on stable Rust.

WonderTix CRM Capstone Project

Link to capstone project's GitHub repo

My capstone project was writing a customer relation management (CRM) web-application for the PortlandPlayHouse. This was a first ever ongoing capstone project that is still being developed by the next capstone teams for the years to come. We were a team of seven students and were proud to be the second team to work on this project. We built the structure, design the UI and the back-end for the CRM. And we also continued building the left-over features from the first team for the ticketing system. At the and of the capstone, we had completed the UI design, database and back-end services. The next step is to create the front-end using ReactJS and HTML/CSS. The project is now becoming open-source on GitHub. We used ReactJS and NodeJS for front-end, PostgreSQL for database, and Typescript for back-end. We also planned to host the site in the near future.
